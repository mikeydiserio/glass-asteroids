## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

3. **Open your browser:**  
   Visit [http://localhost:3000](http://localhost:3000) to play.

## Controls

- **Arrow Keys:** Move and rotate your ship
- **Spacebar:** Shoot bullets
- **R:** Restart after Game Over
